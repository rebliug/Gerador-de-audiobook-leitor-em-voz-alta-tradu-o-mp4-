# Gerador-de-audiobook-leitor-em-voz-alta-tradu-o-mp4-
Basta enviar um arquivo PDF, TXT ou EPUB, e o código irá lê-lo automaticamente, identificar o idioma e oferecer a opção de traduzir o conteúdo para outro idioma. Também funciona com MP3 e MP4 (neste caso, apenas a capa/thumbnail será analisada).
